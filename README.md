## ‚ú® Overview
Lingua Sona literally means **‚ÄúLanguage of Wisdom‚Äù** or **‚ÄúLanguage of Knowledge.‚Äù**  
It is designed to be concise, phonetic, and globally approachable‚Äîcapable of being used for **everyday conversation, programming, mathematics, and formal documentation** alike.

Even though you may notice an alphabet of 251 letters. These letters are assembled as two glyphs interconnected to form each phoenetic used. 
This means instead of learning how to write 251 letters. 
The learner needs only learn how to depict 28 glyphs to write most words. 
With declaring foreign pronounciations and including a few extensions, this count only raises to 31. 
This provides for a large alphabet which can be written quickly and can be used to massively reduce the number of letters needed to write most words.

## üß¨ Core Inspirations

Lingua Sona draws linguistic and structural inspiration from:
- **French** ‚Üí sentence flow & grammar  
- **Swahili** ‚Üí logical morphology, CV structure  
- **Toki Pona / Toki Ma** ‚Üí conceptual grounding & simplicity  
- **Japanese** ‚Üí phonetic vowel system & universal pronounceability  

Originally named **Toki Sona**, the project grew beyond its roots into a language of its own design.

- üí¨ Require **evidentiality** (source of information) for factual statements  
- ‚úçÔ∏è Optimize **writing density**‚Äîmore meaning in fewer characters  
- üß† Use symbols that **mirror pronunciation and mouth shape where possible** for intuitive learning and **require only one to few strokes to write out** for rapid transcription

## Lingua Sona Letter Inventory

Lingua Sona letters are built from five legal structures:

- **CV** ‚Üí consonant + vowel  
- **¬¨V** ‚Üí vowel-initial letter (negated consonant)  
- **C¬¨** ‚Üí consonant-final letter (negated vowel)  
- **CVV** ‚Üí consonant + vowel cluster 
- **¬¨VV** ‚Üí vowel-cluster-initial letter (negated consonant)

All letters use the logical negation symbol **¬¨** as part of the glyph when a consonant or vowel segment is missing.
