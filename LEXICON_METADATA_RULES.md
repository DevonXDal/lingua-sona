# Lexicon Metadata Rules

- One Lingua Sona term = one canonical definition
- Translations are approximate and optional
- No new root may be added without passing the **Root Eligibility Test** found under ~/Language/09_ForAdvancedLinguisitcAndProgrammaticUse/RootEligibilityTest.md
- Affective primitives must be declared explicitly
- Compounds do not count as terms
