# Evidentiality Signifiers.
## What and Why
Evidentiality is mandatory for factual claims it follows the claim as a evidentiality signifier followed by the source of the information when relevant. It is a core part of Lingua Sona and its intent is to back the factual claim by mandating the providing of source information as possible for the claim to have any real backing.

## Format
(Factual Claim) + (Evidentiality Signifier) + (Source - Where Relevant)

## Signifier List
| Signifier | Type                          | Meaning / Use Case                                                            | Example Translation Style                            |
| --------- | ----------------------------- | ----------------------------------------------------------------------------- | ---------------------------------------------------- |
| **TA**    | **Direct Evidence**           | "I personally observed it"                                                    | I saw it.                                            |
| **TE**    | **Indirect / Inferential**    | "I deduced this from signs, not direct experience"                            | I figured it out from clues.                         |
| **TI**    | **Reported / Quoted**         | "Someone else told me this"                                                   | It is hearsay.                                       |
| **TO**    | **Common Knowledge**          | "Everyone knows this / obvious fact"                                          | It is general understanding.                         |
| **TAU**   | **Assumed / Logical**         | "It must be true based on reasoning"                                          | It logically follows.                                |
| **TOOO**  | **Personal Opinion (Low)**    | "In my opinion (uncertain / humble)"                                          | I think so, but I’m not sure.                        |
| **TOU**   | **Personal Opinion (Strong)** | "In my opinion (strong / confident belief)"                                   | I believe this firmly.                               |
| **TEI**   | **Divine/Mystical**           | "It came from the Holy Text. In the words of our God"                         | It is stated by sources I or others view to be holy. |
| **TAI**   | **Calculated Source**         | "It was what was generated by an AI tool or similar means"                    | It came from a calculated source.                    |
| **TON**   | **Narrative Frame**           | "Its a story. If the story says it is true for its internal world then it is" | It is true *within* a story                          |
| **TAW**   | **Hypothesis**                |                                                                               |                                                      |
| **T¬**    | **No Evidential Given**       | Used to negate evidentiality, implying subjectivity                           | Used in poetry or subjective art.                    |
